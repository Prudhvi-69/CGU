react-router-dom : It is a tool which will provide us with the navigation functionalities, functions, tags and etc.,

<BrowserRouter> : Which will let the browser know that React is going to take control of the browser.

<Routes> : This will be containing multiple individual routes, their web url paths and holding them together in one place for easy access.

<Route> : This is an individual route, web url path redirecting to a specific file or location.

<Link> : This is going to create the path for any text, and linking the text to the web url path.
