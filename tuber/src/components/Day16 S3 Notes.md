componentDidMount() : This function will be automatically called when the component gets loaded on the webpage. It will be executed only once, that is when the component is loaded at the beginning.

componentDidUpdate() : This function will be automatically called whenever there is an update inside the component, even if it is a small character change or number change. 

componentWillUnmount() : This function will be called when the component is about to be deleted from the DOM model and from the webpage. Executed only once before the deletion.

**** All these three functions are imported from the built-in class known as : Component which is being extended by our class component.
