React Router DOM : It is a tool which provides the packages that are useful navigation inside a React App.

** It will provide multiple XML system defined tags like : <BrowserRouter>, <Routes>, <Route>, <Link>

<BrowserRouter> : It will let the browser know that we are letting the React take control of the browser with the tags we are going to use inside this Tag.
** All routing related tags should be present inside this one.

<Routes> : It is going to contain multiple single Route and their paths in one place together.

<Route> : It is a single path that any link can be attached to it.

<Link> : This tag is responsible for giving the links between the route (the component) and the browser url path.
